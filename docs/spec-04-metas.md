# Spec 04 — Metas financeiras

## O que faz

- Cadastro de meta: nome, valor, já guardado, data desejada (opcional)
- Cálculo automático: **faltam X**, **Y meses**, **previsão de data**
- Usa a **sobra mensal real** do dashboard (fixos + variáveis + dívidas + cartões)
- Sugestões: guardar +R$ 50/100/200 a mais por mês
- Barra de progresso (% da meta)
